# 60 — Roadmap

Where we're going and what's in flight.

| File | What it covers |
|---|---|
| [`current-quarter.md`](current-quarter.md) | What we're focused on right now |
| [`backlog.md`](backlog.md) | Things we want to do eventually, not yet scheduled |
| [`open-questions.md`](open-questions.md) | Unresolved strategic and technical decisions |
| [`done/`](done/) | Shipped work, archived for reference |

## Philosophy

- **Current quarter is short and opinionated.** If everything is priority, nothing is.
- **Backlog is a wish list.** Don't treat it as a commitment.
- **Open questions are valuable.** Writing them down forces us to pick one — pretending we already decided is worse.
