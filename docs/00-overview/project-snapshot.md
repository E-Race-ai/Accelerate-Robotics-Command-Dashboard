# Project Snapshot

A one-page picture of Accelerate Robotics — what we're building, why, where we are, and what ships next. Refresh this at the start of each quarter.

## Elevator pitch

**Accelerate Robotics** is a hospital robotics OS — one brain, many bots. We build the platform layer that coordinates multi-vendor robot fleets (cleaning, delivery, transport, sensing, and eventually humanoids) inside hospitals and hospitality. Customers avoid vendor lock-in and get a single operational surface across every robot on the floor.

See [`../10-strategy/thesis.md`](../10-strategy/thesis.md) for the full thesis.
