📍 Connect Find My to the dashboard

The dashboard reads your location from the tesla-mobile-office bridge running on your laptop. One-time setup:

  1. Install tesla-mobile-office on your Mac.
  2. Add to its .env: 
    APPLE_ID=ben@accelerate-robotics.com
APPLE_APP_PASSWORD=xxxx-xxxx-xxxx-xxxx
AR_BASE_URL=https://command.accelerate-robotics.com
AR_USER=ben@accelerate-robotics.com
AR_PASSWORD=your-AR-password
  3. Run npm start. The bridge will start polling iCloud and POSTing locations to AR.
  4. Refresh this page. The button will turn 🟢 green when the first heartbeat lands.

App-specific password: generate one at appleid.apple.com → Sign-In and Security → App-Specific Passwords.
