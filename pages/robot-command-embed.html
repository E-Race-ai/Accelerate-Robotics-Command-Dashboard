Robot Command needs home-dashboard running

This tool embeds home-dashboard from your local machine on port 3100. It looks like it's not reachable right now.

Start it locally:

cd ~/Code/home-dashboard && npm run dev

Then reload this page. Once the dashboard is serving on localhost:3100, the Robot Command panel will appear here.

Why this is local-only: home-dashboard runs on Eric's MacBook by default. To use Robot Command from another machine, run home-dashboard there too — or ask Eric for a tunnel URL.
